== Description ==

The Grocery Marketplace is a multipurpose theme designed for creating online grocery stores and supermarket websites, suitable for local shops, mini marts, specialty food stores, and wholesale grocery businesses. It features a clean and structured layout focused on usability and clear product presentation for both retail and wholesale customers. The theme is fully responsive, ensuring consistent display across desktops, tablets, and mobile devices, and is built with an organized code structure to support stable performance and efficient loading. Navigation is designed to be straightforward, making product browsing simple and accessible. It includes customization options that allow adjustments to layouts, colors, and sections through standard controls without requiring advanced technical skills. Banner sections and shortcode support are available for displaying featured products, offers, or informational content, along with a call-to-action button for directing users toward actions such as purchases or inquiries. Additional sections for testimonials, team details, and social media links support content organization and brand communication, while translation readiness makes the theme suitable for multilingual and international use.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Grocery Marketplace in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
